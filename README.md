# SafePay – UPI Fraud Risk Management System using AI

## 📌 Project Overview

SafePay is an AI-powered web application developed to detect and manage UPI fraud risks through intelligent transaction analysis and risk prediction techniques. The project focuses on improving digital payment security with a user-friendly and modern interface.

---

## 🚀 Features

* 🤖 AI-powered risk scoring using Random Forest
* 🔐 Biometric authentication (fingerprint simulation)
* 📷 QR code scanner simulation
* 📊 Circular fraud risk meter gauge
* 💰 Manual balance management
* 📜 Transaction history with search and filters
* 🔔 Notification center
* 👥 Recent payees management
* 📤 Request money feature
* 🌙 Dark mode support
* 🌐 Multi-language support (English/Hindi)

---

---

## ⚙️ Setup Instructions

### 1. Install Backend Dependencies

```bash
cd safepay-app/backend
pip install -r requirements.txt
```

### 2. Start Backend Server

```bash
python app.py
```

### 3. Run Frontend

Open `index.html` in your browser or use Live Server in VS Code.
