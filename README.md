# SafePay - UPI Fraud Prevention System

## AI-Based Real-Time Fraud Detection for UPI Payments

### Features
- 🤖 AI-powered risk scoring using Random Forest
- 🔐 Biometric authentication (fingerprint)
- 📷 QR code scanner (simulated)
- 📊 Circular risk meter gauge
- 💰 Manual balance management
- 📜 Transaction history with search/filters
- 🔔 Notification center
- 👥 Recent payees
- 📤 Request money
- 🌙 Dark mode support
- 🌐 Multi-language (English/Hindi)

### Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** Python Flask
- **ML Model:** Random Forest (scikit-learn)
- **Storage:** LocalStorage (frontend) + In-memory (backend)
